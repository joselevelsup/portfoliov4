Fast Inventory


Client

None - Myself

Technologies

MongoDB, React, React Native, QRCodes, Redux, Express.js, Node.js

Quick Summary

This project started off as a hackathon idea. A friend and I had come up with this idea then after the hackathon, we continued to work on it. This project was meant to help "Mom and Pops" shops with inventory. The mobile app companion would scan the item and update the time the item was updated. The web app side worked like an admin panel. It was meant for the users to see the statistics of the amount of items and which room had been last updated or scanned.


What Was I Able To Accomplish?

I was able to make the mobile app scan the item's QR code and update the timestamp on the web app, CRUD-ing items and rooms, and doing some data visualizing the items per room.


What Did I Learn?

I learned a little more about data visualizing.
